# 📦 Sistema de Inventario — Spring Boot

Sistema de gestión de inventario desarrollado con Java y Spring Boot.

## 🚀 Tecnologías utilizadas

- Java 25
- Spring Boot 4.0.7
- Spring Data JPA / Hibernate
- MySQL
- Maven
- HTML + CSS + JavaScript

## ✅ Funcionalidades

- Login seguro con usuario y contraseña
- Agregar productos con nombre, cantidad y stock mínimo
- Sumar y retirar productos con nombre del responsable
- Estado automático — Disponible, Bajo Stock, Agotado
- Buscador de productos en tiempo real
- Estadísticas en tiempo real — Total, Disponibles, Bajo Stock, Críticos
- Historial de movimientos con fecha y responsable
- Eliminar historial de movimientos
- Cerrar sesión
